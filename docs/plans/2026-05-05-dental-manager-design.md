# 🦷 口腔简易病历管理软件 (Mac M1) 设计方案

- **Date**: 2026-05-05
- **Platform**: Mac (Apple Silicon Native)
- **Status**: Validated

## 1. 核心目标
为小蓝打造一款极致简易、高效且美观的本地化口腔病历管理工具。强调隐私安全与极速录入体验。

## 2. 技术栈
- **核心框架**: Electron + Next.js 15 (React 19)
- **样式**: Tailwind CSS (Apple Bento Grid Style)
- **数据库**: SQLite (Better-SQLite3)
- **打包**: Electron Forge (Universal binary for M1/M2/M3)

## 3. 核心功能
### 3.1 患者检索 (Global Search)
- 左侧常驻搜索框。
- 支持姓名、手机号、姓名拼音首字母实时过滤。
- 列表显示最近就诊患者，点击即刻切换。

### 3.2 极简录入 (Record Editor)
- **患者信息**: 姓名、电话、支付偏好（线上/现金）。
- **2x2 牙位图**: 纯文字网格，对应四个象限，支持 Tab 键快速跳转。
- **诊疗字段**: 主诉 (CC)、检查 (PE)、诊断 (Dx)、处置 (Rx)。

### 3.3 智能短语库 (Quick Snippets)
- **快捷录入**: 输入 `/` 触发浮窗，选择预设短语。
- **自定义管理**: 独立的设置页面，支持用户手动增加、编辑、删除常用术语。

### 3.4 数据管理
- **自动保存**: 停止输入后 2 秒自动同步至本地数据库。
- **PDF 导出**: 一键生成美观的 A4 尺寸病历单。

## 4. 视觉设计 (UI/UX)
- **风格**: 现代苹果风 (Minimal Clinical White)。
- **布局**: 左侧磨砂玻璃导航栏 + 右侧大圆角主操作卡片。
- **交互**: 丝滑的列表位移动画，支持全键盘流式操作。
